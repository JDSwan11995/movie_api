Documentation Page

This application is intended to allow users to find information about several Studio Ghibli movies. Users should be able to sign up and alter profiles including a list of favorite movies.


Request URL HTTP Method Type Request Body Format Response Body Format
Return a list of all included movies /movies This is a GET Request No Request Body "/movies" will retrun a JSON array of all included movies.
Return a specific movie by title. /movies/:title This is also a GET Request No Request Body /movies/:title will return a JSON object with the details of the specific movie.
Return details about a specific included genre /movies/genres/:genreName This is also a GET Request No Request Body /movies/genres/:genreName will return a JSON object with details about the specific genre.
Return details about a director /movies/directors/:directorName This is also a GET Request No Request Body /movies/directors/:director will retrun a JSON object about the specified director.
Create a new user /users This is a POST Request The expected Request Body should be a JSON object structured like this. 
{ name: 'Albert Newman', favoriteMovies: [],}
If the name field is filled and the favoriteMovies field contains a movie on the list then a new user is created.
Update User by ID /user/:id This is a PUT Request The expected Request Body should be a JSON object structured like this. 
{ name: 'Alice Newman', favoriteMovies:[],}
 If the ID number is in the system and the fields are filled out correctly a message will be sent showing the updated data.
Add a movie to the Favorites list /users/:id/:movieTitle This is also a POST Request The expected Request Body should be a JSON object structured like this. 
 { title: [] }
 If the User ID and the movie title exist, the user will recieve a message letting them know it has been added to their favorites list.
Remove a movie from the favorites list /users/:id/:movieTitle This is a DELETE Request No Request Body If the User ID exists, the user will recieve a message letting them know it has been removed from their favorites list.
Delete User /users/:id This is also a DELETE Request No Request Body If the user exists, they are removed and sent a message confirming the removal.
Genre descriptions taken from IMDb.
Usernames taken from Sir Terry Pratchett's Discworld series.
